## 🏗️ 場景一：我想知道某個料件在哪裡？ (看板搜尋)
當現場回報找不到料件，或您需要確認庫存分佈時：

1. **進入地圖**: 點擊左側導覽列的「總覽 (地圖)」。
2. **切換樓層**: 在畫面左上方，可以看到「新大樓3樓」、「新大樓4樓」等按鈕，點擊即可切換不同的區域。
3. **搜尋料號**: 在畫面右上角的搜尋框，輸入關鍵字（例如料號的前幾碼）。
4. **辨識標示**:
   * **紅色光圈**: 代表該料件目前所在的「儲位」。
   * **綠色儲位**: 代表該位置目前有存放東西。
   * **滑鼠操作**: 將滑鼠移到儲位格子上，會自動跳出小視窗，顯示該位置的「詳細料號、品名與剩餘數量」。

## 📦 場景二：我要準備週報或領料清單？ (Excel 報表)
當您需要導出完整的庫存統計給主管或是進行月盤點時：

1. **進入報表**: 點擊左側「庫存報表」。
2. **選擇分頁**: 畫面上方有四個分頁，最常用的是：
   * **料件總表**: 以「料號」為主角，看每個料號總共剩多少、散佈在哪裡。
   * **儲位總表 (最推薦)**: 以「位置」為主角，看每一層樓的貨架上放了什麼。
3. **匯出 Excel**: 
   * 點擊右上角綠色的 **「匯出 Excel」** 按鈕。
   * **專業功能**: 系統會自動幫您下載一份檔案，裡面已經根據**樓層**分好了不同的「分頁 (Sheets)」，您不需要再手動篩選樓層，直接列印即可。

## 📥 場景三：公司進了大量新貨，要更新資料？ (批量匯入)
當有幾百筆資料要更新，手動輸入太慢時：

1. **進入匯入**: 點擊左側「資料匯入」。
2. **使用範本**: 
   * 請確保您的 Excel 欄位與系統要求的「特定格式」一致。
   * 您可以參考 `文件/20260210 料架QRcode.xlsx` 這個檔案的格式。
3. **上傳檔案**: 點選「選擇檔案」後按上傳。
4. **確認訊息**: 系統會顯示「匯入成功 X 筆」，這代表伺服器已經即時更新，此時再去地圖看，資料就已經變新了。

## 📱 場景四：到現場協助盤點或移庫？ (手機模式)
如果您拿著平板或手機在倉庫現場：

1. **切換模式**: 點擊左下角的「切換至手機掃描模式」。
2. **掃描操作**:
   * **第一步**: 掃描料件上的 QR Code (或手動選取料號)。
   * **第二步**: 掃描儲位上的貼紙。
   * **第三步**: 確認增加或減少的數量，然後按「送出」。
3. **紀錄同步**: 您在手機按下的每一筆，辦公室主機會立刻更新，不需要再回傳資料。

## ❓ 常見問與答 (FAQ)
* **Q: 畫面卡住了怎麼辦？**
  * A: 請按一下鍵盤最上方的 `F5` 重新整理網頁即可。
* **Q: 匯出 Excel 時，檔案名稱有亂碼？**
  * A: 檔案名稱會包含當天的日期（如 20260327），如果瀏覽器詢問存檔位置，您可以自行命名方便尋找。
* **Q: 為什麼地圖上看不到剛匯入的東西？**
  * A: 請檢查該樓層切換是否正確，或是料件的數量是否大於 0（系統預設只會亮起有數量的儲位）。
