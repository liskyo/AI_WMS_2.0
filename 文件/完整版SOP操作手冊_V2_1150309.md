# 智慧倉儲出入料系統 - 完整版SOP操作手冊 V2 📖

歡迎使用智慧倉儲系統！本手冊將以最簡單的圖文步驟，帶領您快速熟悉系統的各項日常操作與全新防呆升級功能。

---

## 登入與首頁認識

### 第一步：登入系統
1. 開啟系統網頁或直接透過**手機瀏覽器**進入。
2. 輸入您的 **員工編號 (帳號)** 與 **密碼**。
3. 點選「登入」。
> 💡 *提示：系統具備**使用者與權限管理**，若您為一般操作員，部分作廢與匯入功能將被限制。忘記密碼請聯繫主管或管理員。*

### 第二步：認識側邊欄選單
登入後，畫面左側會有一排選單，這是您每天會用到的功能：
1. 🏠 **總覽看板**：看地圖、看整體庫存概況、查看**安全庫存警示**。
2. 🔄 **出入庫作業**：跨平台日常撿料、入庫、BOM 表整批出庫。
3. 🕒 **出入庫紀錄**：查過去的領料紀錄、操作員軌跡、抓錯作廢（需權限）。
4. 📦 **庫存查詢**：快速精準搜尋特定的料件與數量儲位。
5. 📊 **庫存報表**：列印所有庫存清單與採購建議。
6. ⚙️ **資料匯入** (限管理員)：更新 Excel 資料與地圖座標的地方。

## 日常操作一：如何找料？ (看地圖與庫儲查詢)

如果您想找某個料號放在哪裡，有下列最快的方法：

### 🎯 方法 1：從「總覽看板」找 (視覺化 2D 地圖)
1. 點選左側選單的 **總覽看板 (地圖)**。
2. 在右上角的搜尋框中，輸入您想找的「料號」或「品名」。
3. 地圖上相對應的「貨架儲位」就會 **閃爍紅燈 (動態定位)**！
4. 滑鼠移過去，會自動顯示該儲位上的「所有疊貨物料」清單與數量。

### 🔍 方法 2：全域「庫存查詢」
1. 點選左側的 **庫存查詢**。
2. 在搜尋欄輸入料號或部分品名。
3. 系統會**秒級定位**並列出該物料目前存放在「哪些儲位」，以及各有多少個，完全取代傳統紙本翻找。

## 日常操作二：智慧手機跨平台掃描出入庫

當您從產線拿走物料，或進貨補上物料時，可直接使用您的手機進行高頻掃描，無需昂貴硬體掃描槍。

1. 打開手機登入系統，點選選單的 **出入庫作業**。
2. 選擇「單筆入庫」或「單筆出庫」分頁。
3. 點擊 **相機圖示 (啟動掃描)**，系統會要求相機權限，請選擇「允許」。
4. 將手機鏡頭對準料件條碼 (Barcode) 或 儲位 QR Code。
5. 掃描成功後，畫面會自動帶入料號或儲位代碼。
6. 輸入**數量**，點擊 **確認提交**。系統會以您的「員工編號」紀錄這筆異動軌跡。

> ⚠️ *防呆小提醒：如果您要「出庫」的數量大於該儲位現有的數量，系統會跳出紅色警告阻擋您。本系統具備連續掃描防呆機制，高頻掃描也不會閃退或白屏！*

---

## 日常操作三：BOM 表一鍵整批出庫

要組裝完整產品，這項功能幫您自動一次展開並把所有子零件的帳扣掉！

1. 點選 **出入庫作業**，切換到 **BOM 領料** 標籤。
2. 輸入 **主件品號** 與預計組裝的 **生產套數** (例如 5 套)。
3. 點選 **查詢展開**。
4. 系統將自動列出所有子零件明細，與建議前往檢料的儲位（優先帶出有庫存的儲位）。
5. 確認無誤後，**一鍵點擊「執行出庫」**，即完成一次性跨儲位整批扣帳。

---

## 日常操作四：安全庫存智能警示與自動採購單

系統會主動幫您顧好庫存的防護網。

1. 進入 **總覽看板 (首頁)**。
2. 若有物料跌破「安全庫存」，首頁畫面上方或顯眼處會自動亮起 **紅色動態警告區塊**。
3. 點擊該紅燈區域，系統會自動幫您過濾出所有「待補貨 / 低水位」的物料清單。
4. 點擊右上方的 **匯出建議採購單 Excel**，系統就會產出一份排版好的表格，可直接交給採購人員！

---

## 異常處理：打錯數量或選錯庫位？ (權限防呆作廢)

為確保每一筆帳務都能精準追蹤 (ACID)，系統提供嚴謹的作廢與自動回滾機制。

1. 點選 **出入庫紀錄**。
2. 找到您剛剛打錯的該筆紀錄，點擊右側紅色的 **[作廢]** 按鈕。
3. 系統將要求驗證 **管理員密碼** (一般操作員無法自行刪帳，需請示主管解鎖授權)。
4. 驗證通過後，該紀錄狀態變更為「已作廢 (Void)」。
5. **系統會完美回滾 (Rollback) 數量**，自動把您不小心扣錯的料加回，或入錯的料扣除，絕不留死板的髒資料。

## 系統管理員： Excel 資料匯入與非同步對接

*(**以下功能僅限具備管理員帳號權限者使用**)*

1. 點選 **資料設定中心 / 匯入中心**。
2. 選擇需要更新的目標模組 (包含：料件主檔、盤點庫存基準、儲位平面圖座標)。
3. 若不清楚欄位，可先點擊 **下載 Excel 範本**。
4. 將準備好的 Excel 檔案拖曳上傳，核對預覽結果後點擊 **確認匯入**。
5. 本系統具備**萬筆秒級覆蓋**機制，若匯入的資料有殘缺報錯，系統將整批退回原狀，避免半殘資料寫入。
6. ***(ERP 對接小知識)***：針對單據拋轉，您只需將 ERP 產出的 Excel 存入指定資料夾，背景服務將自動幫您讀取並列為待發單據，無需您按任何匯入按鈕。
